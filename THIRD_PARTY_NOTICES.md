# Third-Party Notices

## Game assets

The MVP uses no externally authored commercial/stock art, audio, or shader assets. No paid asset pack was added. The bilingual mission briefing embeds the open-source font subset documented below.

### AI-generated project sources

The following source images were supplied by the project maintainer as AI-generated concept material and are configured inputs to the reproducible runtime-art pipeline:

- `assets/source/generated/player_animation_source.png`
- `assets/source/generated/guard_animation_source.png`
- `assets/source/generated/facility_map_tileset_source.png`

`assets/source/generated/facility_map_reference.png` is a separate 824×807 AI-generated topology and art reference. It is preserved unchanged, is not an input to `process-all`, and is not shipped as a gameplay texture; the runtime facility is rebuilt from project-authored TileMap layers, scenes, and blueprint data.

An additional AI-generated player concept is retained under `assets/concept/` as an unused reference. No separate artist, stock-library, paid-pack, or external-game attribution was supplied for these images. The generating model/service, prompts, generation identifiers, and service terms were also not supplied in repository metadata, so this notice does not claim provenance details that cannot be verified. If that metadata becomes available, it should be added here.

`assets/source/environment/helix_environment_concept_v1.png` is a separate project-generated AI direction board created during the HELIX environment pass. It contains no downloaded stock or third-party game art and is retained only as a design reference. The runtime atlas is independently and deterministically drawn by `tools/environment_art_pipeline.py` from the project's palette/material specification; it does not crop or sample pixels from that board. The direction board is excluded from Web and desktop exports by `export_presets.cfg`.

Runtime derivatives are reproducibly generated by `tools/asset_pipeline.py` and `tools/environment_art_pipeline.py`. Gameplay references only derivative atlases in `assets/sprites/` and project-authored Godot map resources. The full concept sheets, the 824×807 facility reference, and review previews are source-only and are never loaded as gameplay textures. See `docs/asset_manifest.md` for hashes, usage, and quality limitations.

If an external asset is added later, its source URL, author, version, license, redistribution terms, and required attribution must be recorded here in the same change.

### Noto Sans KR UI subset

## Build-only dependencies

GitHub's official Actions used by the workflows are pinned to immutable commit SHAs. These build tools are not authored by this project and are not game content.
